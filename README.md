# Приложение «Products App»

Одностраничное приложение (SPA) на Next.js (App Router) с TypeScript и Redux Toolkit для отображения списка товаров с возможностью лайка, удаления, фильтрации и просмотра детальной страницы.

## Стек
- Next.js 16 (App Router)
- React 19
- TypeScript 5
- Redux Toolkit + React Redux
- Tailwind CSS
- Lucide React (иконки)

## Функциональность
- Главная страница `/` — список товаров
- Страница создания товара `/create-product` — форма на английском, сохранение в общее хранилище
- Лайк товара (переключение + подсветка иконки)
- Удаление товара
- Фильтр: «Все» и «Избранное»
- Обрезка текста карточек для единой высоты
- Детальная страница товара по адресу `/products/[id]`
- Все полученные и созданные данные хранятся во внутреннем Redux store

## Используемый интерфейс данных
Для получения данных используется «FakeStoreAPI»:

- **Адрес сервиса**: https://fakestoreapi.com/
- **Документация сервиса**: https://fakestoreapi.com/docs
- **HTTP‑запрос**: `https://fakestoreapi.com/products`
- **Структура ответа**: массив объектов с полями `id`, `title`, `description`, `price`, `image`, `category`

**Резервные данные**: Если внешний сервис недоступен (ошибка сети, ограничения межсайтовых запросов и т.п.), приложение автоматически использует встроенные мок‑данные на русском языке. Это обеспечивает работу приложения даже при проблемах с доступом к удалённому сервису.

Приложение приводит ответ сервиса к внутренней модели и использует поле `image` как картинку товара. Товары без полных данных (название, описание, цена, изображение) автоматически фильтруются и не отображаются.

## Скрипты
- `npm run dev` — запуск в режиме разработки
- `npm run build` — сборка в продакшен


## Структура
- `app/page.tsx` — главная страница (список товаров)
- `app/products/[id]/page.tsx` — детальная страница товара
- `lib/slices/productsSlice.ts` — слайс Redux для товаров (загрузка данных, лайки, удаление, фильтр)
- `lib/store.ts` — конфигурация хранилища Redux
- `lib/hooks.ts` — типизированные хуки
- `components/ProductCard.tsx` — карточка товара
- `components/ProductList.tsx` — список товаров и фильтрация
- `components/Filter.tsx` — фильтр
- `components/ReduxProvider.tsx` — провайдер Redux

## Запуск
1. Установите зависимости: `npm install`
2. Запустите dev-сервер: `npm run dev`
3. Откройте `http://localhost:3000`
